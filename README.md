# School Schedule App

This is a basic Node.js Express project demonstrating how to create a simple school schedule API and write unit tests using Jest and Supertest.

**Features:**

- Fetches a sample school schedule.
- Includes basic unit tests to ensure the API functionality.

**Getting Started:**

1. Clone the repository: `git clone <repository_url>`
2. Install dependencies: `npm install`
3. Start the server: `npm start`
4. Run tests: `npm test`

**Endpoints:**

- `/schedule`: Returns a JSON array of the school schedule.
